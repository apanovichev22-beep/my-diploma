Краткое описание: Разработан гибридный метод обнаружения сетевых аномалий для корпоративных сетей, объединяющий ансамбль алгоритмов классификации (Random Forest + Decision Tree) с ассоциативными правилами Apriori. Метод позволяет снизить долю ложных срабатываний (FPR) на **44,5%** без потери полноты обнаружения атак.
